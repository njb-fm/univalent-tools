# univalent-tools
-----

## なにこれ
えーなんだろこれ。簡単に言うとBashとかで動くシステム設定用ソフトでーす。
Alter Linuxのi3-wm版に搭載されているヤツに影響されて「こんなのがあるといいなー」みたいな考えで作りました。

以下の事を行なえます。

- ネットワーク設定 (Network Manager)
- 音声設定 (Pavucontrol{-Qt})
- パッケージ管理 (Pamac)

## 依存パッケージ
- zenity
- nm-connection-editor
- pavucontrol / pavucontrol-qt
- pamac
